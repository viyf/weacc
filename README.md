## 可用脚本

### Bilibili 可访问性优化

主要解决了以下问题：

 * “点赞”、“投币”、“收藏”、“分享” 按钮缺乏键盘操作能力；
 * “搜索” 按钮无替代描述；
 * 弹幕、 CC 字幕没有添加 ARIA Live Region 相关属性；
 * “弹幕” 开关无替代文字描述；
 * “发送” 弹幕按钮无键盘聚焦能力；
 * 当播放器组件加载完成后 “播放”/“暂停” 按钮没有获取焦点；
